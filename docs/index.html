Gabe's Awesome Skills

A curated collection of reusable AI agent skills. Each skill is a self-contained folder your agent loads on demand—no framework, no lock-in, just sharp instructions that work.

Browse Skills Create a Skill

How it works

One folder. One file. Infinite capability.

Skills are knowledge packages that teach AI agents how to do things better. Agents see only a name and description until they need the full instructions—progressive disclosure keeps context lean.

⚡

Portable

No framework dependencies. Works with Claude Code, OpenCode, Codex, Cursor, Copilot, and more.

📦

Self-contained

Each skill lives in its own folder with scripts, references, and assets colocated. Easy to share, copy, or install.

✂

Minimal by design

No bloat, no boilerplate. Every token in a SKILL.md earns its place. Scripts over essays, examples over explanation.

Anatomy

Skill structure

skills/my-skill/
├── SKILL.md          # required — agent instructions
├── scripts/          # optional — helper scripts
├── references/       # optional — docs loaded on demand
└── assets/           # optional — templates, icons, etc.

The only required file is SKILL.md with YAML frontmatter containing name and description. Everything else is optional and loaded only when needed.
